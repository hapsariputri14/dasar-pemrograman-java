# ☕ Java Programming Fundamentals: Academic Coursework

Repositori ini adalah dokumentasi lengkap perjalanan belajar saya dalam mata kuliah **Dasar Pemrograman**. Berisi implementasi kode Java yang mencakup pemahaman algoritma, struktur data dasar, hingga pemrograman modular.

---

## 📂 Ringkasan Materi & Struktur Folder

Setiap folder berisi kumpulan latihan, tugas modifikasi, serta studi kasus yang dikerjakan menggunakan **Java**.

### 🔹 Dasar-Dasar & Alur Logika (Week 2 - 5)
Fokus pada pemahaman sintaks dasar dan alur keputusan.
* **`daspro-jobsheet2` & `3`**: Implementasi variabel, tipe data, dan operator dasar (Contoh: Kalkulator bunga bank, perhitungan harga bayar).
* **`daspro-jobsheet4`**: Perancangan alur menggunakan **Flowchart** dan implementasi pseudocode ke dalam kode Java.
* **`daspro-jobsheet5`**: Logika pemilihan dasar menggunakan `if-else` dan `switch-case`.

### 🔹 Pemilihan & Perulangan Lanjutan (Week 6 - 8)
Fokus pada kontrol alur yang lebih kompleks.
* **`daspro-jobsheet6`**: Pemilihan bersarang (Nested Selection) untuk studi kasus yang lebih mendalam.
* **`daspro-jobsheet7`**: Implementasi perulangan menggunakan `for`, `while`, dan `do-while` (Contoh: Sistem cuti, perhitungan gaji).

### 🔹 Struktur Data Array & Nested Loops (Week 9 - 11)
Fokus pada pengelolaan data berkelompok dan pola logika.
* **`daspro-jobsheet9`**: Dasar Array 1-Dimensi (Contoh: Linear search, pencarian nilai rata-rata).
* **`daspro-jobsheet10`**: Implementasi Array Multi-Dimensi (Studi Kasus: Sistem manajemen kursi bioskop).
* **`daspro-jobsheet11`**: Penerapan **Nested Loops** untuk pembuatan pola (Segitiga angka, persegi angka, pola bintang).

### 🔹 Pemrograman Modular (Week 13+)
Fokus pada pengorganisasian kode agar lebih efisien.
* **`daspro-jobsheet13`**: Penggunaan **Fungsi (Method)** untuk membuat kode yang *reusable* dan terstruktur (Contoh: Fungsi ucapan terima kasih, modifikasi percobaan fungsi).

---

## 🛠️ Kemampuan Teknis yang Ditunjukkan

Melalui repositori ini, saya mendemonstrasikan pemahaman dalam:
- **Problem Solving**: Mengonversi masalah dunia nyata (seperti sistem gaji atau reservasi bioskop) menjadi kode.
- **Code Refactoring**: Melakukan modifikasi pada kode yang sudah ada untuk meningkatkan fungsionalitas.
- **Logika Algoritma**: Mahir dalam penggunaan *Nested Loops* dan *Complex Conditionals*.
- **Data Management**: Mengelola koleksi data menggunakan Array secara efektif.
