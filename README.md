# README for Dasar Pemrograman Java

## About Repository
This repository serves as a comprehensive resource for learning the fundamentals of Java programming. It includes various coding exercises and examples tailored for beginners.

## Learning Objectives
- Understand the basics of Java syntax and semantics.
- Develop problem-solving skills by implementing various programming practices.
- Gain familiarity with Java development tools and libraries.

## Jobsheet List
| Number | Title                         | Description                            |
|--------|-------------------------------|----------------------------------------|
| 1      | Introduction to Java          | Basic syntax and structure of Java     |
| 2      | Control Structures            | Implementing if-else and loops         |
| 3      | Object-Oriented Programming   | Classes, objects, and inheritance      |
| 4      | Exception Handling            | Understanding and managing exceptions   |

## How to Use
1. Clone this repository to your local machine.
2. Open the project in your preferred IDE (e.g., IntelliJ, Eclipse).
3. Follow the instructions provided in the respective jobsheets.

## Prerequisites
- Java Development Kit (JDK) installed on your computer.
- Basic understanding of programming concepts.
